## 📈 Key Insights
- West region has the highest sales.
- North region has the lowest sales.
- Electronics category is most profitable.
- Sales peak in certain months.
- Total Sales: 51.74M
- Total Profit: 7.90M
- Profit Margin: 15.27%

## 📊 Visuals Used
- KPI Cards (Sales, Profit, Orders, Margin)
- Bar Chart (Region Analysis)
- Line Chart (Monthly Trend)
- Pie Chart (Payment Mode)
- Category Profit Analysis
